Dino Think

is a fun application that guides the user through the Design Thinking process. The Design Thinking process is a process for rapid prototyping that helps users pinpoint and solve problems. This is done by guiding users through timed steps that are built to keep users in a quick thinking intuitive state of mind. Creativity is encouraged no matter how far-fetched, unbelievable, crazy, ridiculous, or awesome it may be!
